# Bridge Evidence Capture Directory

Status: local generated evidence directory

This directory is reserved for timestamped emulator-only readonly bridge capture outputs generated by:

```powershell
powershell -ExecutionPolicy Bypass -File .\rom_lab\bridge\capture_readonly_bridge_evidence.ps1 -Mode identity
```

Generated `*.json` and `*.md` capture files in this directory are ignored by Git by default.

Commit summarized status reports under `reports/` instead of raw per-run capture artifacts unless a future review explicitly decides that a specific sanitized capture should become durable repo evidence.

Safety posture:

- emulator-only
- read-only
- no physical phone
- no firmware acquisition
- no stock image acceptance
- no recovery anchor acceptance
- no raw ADB or fastboot passthrough
